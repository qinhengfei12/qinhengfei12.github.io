SqlMap的使用

简介

SQLMap是一个自动化的SQL注入工具，主要的功能是扫描，发现并利用给定的URL的注入漏洞，内置了很多的绕过插件，支持MYSQL等常见的数据库

​ 他的注入技术包括 :

​ 1. 基于布尔类型的盲注

​ 2. 基于时间的盲注

​ 3. 基于报错的注入

​ 4. 联合查询注入

​ 5. 堆查询注入

安装

sqlmap下载地址：http://sqlmap.org/

需要的python环境

Python下载地址：https://www.python.org/

​ 需要的是python2.7的环境

!!注意事项

1． 将python安装在C盘下，配制python的环境变量，将splmap解压在python的根目录下。

2． 在sqlmap的目录下创建一个文件 sqlmap.bat ，将其中的内容编辑成 cmd.exe

保存，双击打开文件

使用sqlmap

1. 在命令行中敲击

python sqlmap.py -h

出现如图，则表示安装成功

-h 是sqlmap的帮助文档

参数解释

–level 5

1.此参数是用于测试等级的,是利用Payload划分出来的

2.此参数一共有5个等级,其在每次的执行中都会应用,其等级的默认为1.

3.其中5等级包含的Payload最多,可以自动破解出cookie和XFF等头部注入.

4.这个参数是可以影响注入点的.

​ –level 2 是可以测试HTTP cookie的

​ –level 3 是可以测试HTTP User-Agent/Referer头

​ 5.从操作方面讲,当不知道时,level的等级越高越好

–is-dba

​ 1.此参数查看当前用户是否为管理员

–roles

​ 列出数据库管理员角色

–referer: HTTP Referer 头

​ 这个参数可以用来伪造referer头,当level的等级是3以上时,sqlmap会对其及进行尝试性的注入.

–sql-shell

​ 运行自定义的SQL语句

–file-read

​ 从数据库服务器中读取文件

–file-writer –file-dest

​ 上传文件到数据库服务器中,可以是文本或二进制文件
